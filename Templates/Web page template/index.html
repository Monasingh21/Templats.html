WEB DEVELOPMENT TASK

Build a Multi-Section Blog Page

Objective: Create a multi-section blog page using HTML to demonstrate your web development skills, including formatting content, creating links, embedding media, and organizing information.

TASK REQUIREMENTS

  • Follow HTML5 structure: <!DOCTYPE html>, <html>, <head>, <body>
  • Use headings, paragraphs, formatting tags, image, table and lists
  • Add links: internal, external, downloadable
  • Use media: audio and video
  • Organize content: welcome section, table, links and media

INSTRUCTIONS

1

Set up the HTML5 structure:
Start with <!DOCTYPE html>, <html>, <head>, <body>. This ensures proper structure in all browsers.

2

Format the Content:
Use headings, paragraphs and formatting tags. Add header, sections, bold, italic, buttons and marquee.

3

Create Links & Lists:
Add lists and links (internal & external). Use ordered, unordered and description lists.

4

Add Images, Tables & Media:
Add image, table, video and audio. Embed media elements to make page interactive.

SUBMISSION CHECKLIST

  • Use proper HTML5 structure
  • Include headings, paragraphs, formatting
  • Add lists, table, image, video, audio
